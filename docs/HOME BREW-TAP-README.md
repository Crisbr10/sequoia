# homebrew-tap

🍺 Homebrew tap for [Crisbr10](https://github.com/Crisbr10) tools.

## Usage

```bash
brew tap Crisbr10/sequoia
brew install sequoia
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `sequoia` | Multi-agent code audit framework with Go CLI installer and TUI |

## How It Works

This repository is automatically updated by [GoReleaser](https://goreleaser.com) on every Sequoia release. Do not edit Formula files manually — they will be overwritten on the next release.
