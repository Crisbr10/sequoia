# Sequoia v0.1.0 — Release Notes

> First public release · 2026-05-10

## What's New

Sequoia v0.1.0 is the first public release of the multi-agent code audit
framework. It includes the full CLI installer, interactive TUI, and adapters
for five AI coding tools.

### Multi-Agent Audits

Deploy 9 specialized agents that inspect your project from every angle:

| ID | Agent | Domain |
|----|-------|--------|
| C0 | Orchestrator | Project map, coordination, synthesis |
| P1 | Security | Auth, injection, secrets, attack surface |
| P2 | Performance | Bundle size, queries, caching, Core Web Vitals |
| P3 | Architecture | Patterns, coupling, API design, tech debt |
| P4 | Quality | Test coverage, naming, dead code, documentation |
| P5 | Experience | Accessibility, UX patterns, conversion flows |
| P6 | Operations | CI/CD, containerization, observability, IaC |
| M1 | Correlator | Cross-phase deduplication and root cause analysis |
| M2 | Reporter | Health Score, deliverables, prioritized action plan |

### CLI Installer

```bash
sequoia install              # Interactive TUI
sequoia install --no-tui     # Headless, all tools
sequoia status               # Check installation state
sequoia uninstall --all      # Remove from all tools
sequoia version              # Print version
```

### Interactive TUI

The Bubbletea-powered TUI guides you through installation with:
- Tool auto-detection and multi-select
- Language and persistence configuration
- Real-time progress per tool
- Error recovery with retry
- Status and uninstall management

## Installation

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/Crisbr10/sequoia/main/scripts/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/Crisbr10/sequoia/main/scripts/install.ps1 | iex
```

### From Source

```bash
git clone https://github.com/Crisbr10/sequoia.git
cd sequoia
go build -o sequoia ./cmd/sequoia/
./sequoia install
```

### Package Managers

```bash
# Homebrew (macOS/Linux)
brew install Crisbr10/homebrew-sequoia/sequoia

# Scoop (Windows)
scoop bucket add sequoia https://github.com/Crisbr10/scoop-sequoia
scoop install sequoia
```

## Known Limitations

- **E2E audit tests pending**: End-to-end validation of actual audit runs
  (T-012, T-017) requires a full Claude Code / OpenCode session and has
  not been automated.
- **TUI English only**: The TUI supports language selection but Spanish
  translations are not yet complete.
- **Plugin system foundation**: Interface and loader are implemented,
  but example plugins need real audit logic. Scheduled for v0.2.0.

## Upgrading

This is the first release. No upgrade path is needed.

For future releases, run the one-line installer again — it detects
your current version and upgrades automatically.
