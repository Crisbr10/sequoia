# scoop-bucket

📦 Scoop bucket for [Crisbr10](https://github.com/Crisbr10) tools.

## Usage

```powershell
scoop bucket add crisbr10 https://github.com/Crisbr10/scoop-sequoia
scoop install sequoia
```

## Available Manifests

| Manifest | Description |
|----------|-------------|
| `sequoia` | Multi-agent code audit framework with Go CLI installer and TUI |

## How It Works

This repository is automatically updated by [GoReleaser](https://goreleaser.com) on every Sequoia release. Do not edit bucket files manually — they will be overwritten on the next release.
